# LLM Error Detection for Math Word Problems

## 📌 Overview
This project evaluates how well large language models (LLMs) can detect errors in math word problems (MWPs) for US primary school (Grades 1–6).

## 📊 Dataset
- 5,098 math word problems  
- 12 error categories  
- Generated using LLMs  

## 🤖 Models
- Mixtral-8x7B  
- Llama-3.1-8B  
- DeepSeek-Math-7B  
- Llama-3.2-3B  

## 🧪 Methods
- Zero-shot  
- One-shot  
- Three-shot  
- Fine-tuning  

## 📈 Results
- Llama-3.2-3B achieved ~90% accuracy in zero-shot  
- Similar performance to larger models with fewer resources  
- Complex prompts reduced accuracy  
- Fine-tuning did not always improve results  

## 💡 Key Takeaways
- Smaller models can perform very well  
- Simple prompts work better than complex ones  
- Data quality has a strong impact on performance  

## 🛠️ Tech Stack
- Python  
- LLMs  
- Prompt Engineering
